# Rsbuild + Tailwind v4 + Tanstack Router Project

## Setup and Getting Started

### Setup

To install the dependencies:

```bash
pnpm install
```

### Get started

To start the dev server:

```bash
pnpm dev
```

To build the app for production:

```bash
pnpm build
```

To preview the production build locally:

```bash
pnpm preview
```
